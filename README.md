# intent-portal

## Description

Minimal Android Intent Portal for JekyllEx and GitHub Pages

Минимальный Android Intent Portal для тестирования intent://, android-app:// и deep links.

## Repository Name

```text
intent-portal
```

```text

```

# Установка

## 1. Установить JekyllEx

https://jekyllex.xyz

или через F-Droid:
https://f-droid.org/packages/xyz.jekyllex/

---

## 2. Создать проект

- New Project
- Blank Project
- Название:
  intent-portal

---

## 3. Скопировать файлы

Вставить содержимое архива в проект.

---

## 4. Запустить сервер

Через кнопку:

```text
▶ Serve
```

или Terminal:

```bash
bundle exec jekyll serve
```

---

## 5. Открыть сайт

```text
http://127.0.0.1:4000
```

---

## 6. GitHub Pages

GitHub → Settings → Pages

Deploy from branch:
- main
- /(root)

---

## 7. Сайт

```text
https://USERNAME.github.io/intent-portal/
```
